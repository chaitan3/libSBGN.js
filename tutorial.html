General Information of libSBGN.js

In general, libSBGN.js is modeled from SBGN(System Biology Graph Notation). All classes and functions of libSBGN.js is prefixed with “sb.”. A system biology graph is represented as a sb.Document. sb.Document contains instances of sb.Node and sb.Arc as well other objects like sb.ArcGroup and sb.Port. You can have a look at the API reference.

Tutorial 3: Reading a file

Currently only SBGN-ML is supported by the file reader. But our goal is to support jSBGN, SBGN-ML, SBML and BioPAX. You can either read a file from a string object which contains the file content or a URL points to the file on the internet. If you use the URL, the file will go through our proxy which wraps the file content with jsonp(to overcome cross domain HTTP request restrictions of web browsers). We will not either log, store, or share the data go through our proxy. And you can easily setup your own proxy as well.
